# ecommerce-system

A microservices-based e-commerce system built with Spring Boot.

## Services

- **order-service** – Handles order creation and retrieval (Port: 8081)
- **inventory-service** – Manages product inventory (Port: 8082)

## Tech Stack

- Java 21
- Spring Boot 4.0.6
- Spring Data JPA
- PostgreSQL
- Lombok
- Maven
